# ✈️ Plan-Go: 실시간 협업 여행 플래너

> **1인 개발 프로젝트** | **웹 & 앱 확장형 아키텍처**
> "Plan Together, Record Forever" (함께 계획하고 영원히 기록하다)

---

## 📌 1. 프로젝트 개요
* **목적**: 복잡한 여행 일정을 지도 기반으로 시각화하고, 동행자와의 실시간 협업을 통해 계획의 효율성 극대화.
* **대상**: 체계적인 여행 계획을 선호하는 자유 여행객 및 기록 공유를 즐기는 사용자.
* **핵심 가치**: 사용자 경험 중심의 UI/UX와 실시간 데이터 동기화 기술 증명.

---

## 🛠 2. 기술 스택 (Tech Stack)

| 구분 | 기술 스택 | 이유 |
| :--- | :--- | :--- |
| **Frontend** | React, TypeScript, Tailwind CSS | 컴포넌트 기반 설계 및 추후 React Native 로직 공유 최적화 |
| **State** | Zustand | 가벼운 상태 관리로 복잡한 일정 로직 처리 |
| **Backend** | Supabase (PostgreSQL, Realtime) | 실시간 동기화(WebSocket) 및 빠른 API 서버 구축 |
| **Maps** | Google Maps API | 글로벌 장소 검색 및 경로 최적화 시각화 |
| **Deployment** | Vercel | CI/CD 파이프라인 구축 및 자동 배포 |

---

## ✨ 3. 핵심 기능 (Core Features)

### 🗓 1단계: 스마트 플래너 (MVP)
* **드래그 앤 드롭 일정 관리**: 장소를 날짜/시간별로 자유롭게 배치 및 순서 변경.
* **지도 연동 시각화**: 추가된 장소들을 마커로 표시하고 이동 경로 자동 시각화.
* **실시간 동시 편집**: 동행자를 초대하여 하나의 계획을 실시간으로 함께 수정.

### 📸 2단계: 여행 기록 (Feed)
* **타임라인 후기**: 일정 단위별 사진 및 텍스트 기록 보관.
* **비용 정산 기능**: 여행 중 발생한 지출 입력 및 인원별 자동 정산.
